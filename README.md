# Open Source Audit — Capstone Project
Open Source Audit Project using Python and Linux shell scripts

## About the Project

This project is part of the Open Source Software course. In this project, I explored the concepts of open-source software by choosing Python as my case study. I studied its origin, license, ecosystem, and how it works in a Linux environment.
Along with theory, I also created and tested multiple shell scripts to understand how open-source tools work practically in a real system.

### Shell Scripts

## Script 1 — System Identity Report

Displays system information like user, kernel, uptime, and date.

## Script 2 — FOSS Package Inspector

Checks if Python is installed and shows its details.

## Script 3 — Disk and Permission Auditor

Analyzes directories, permissions, and disk usage.

## Script 4 — Log File Analyzer

Reads a log file and counts keyword occurrences.

## Script 5 — Open Source Manifesto Generator

Generates a personalized manifesto using user input.

## How to Run Scripts

```bash
chmod +x script1.sh
./script1.sh

chmod +x script2.sh
./script2.sh

chmod +x script3.sh
./script3.sh

chmod +x script4.sh /var/log/syslog

chmod +x script5.sh
./script5.sh
'''
